# Bike Sales Excel Dashboard

## 📌 Project Overview
This project is an interactive Excel dashboard created to analyze bike sales data. It uses Pivot Tables, Pivot Charts, and Slicers to provide insights into customer demographics and sales trends.

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning

## ✨ Features
- Interactive dashboard
- Sales analysis by region
- Customer age analysis
- Income analysis
- Gender comparison
- Easy filtering using slicers

## 📂 Files
- `Bike Sales - Excel Dashboard Project.xlsx` – Main Excel dashboard
- `README.md` – Project documentation

## 🚀 How to Use
1. Download the Excel workbook.
2. Open it in Microsoft Excel.
3. Use the slicers to interact with the dashboard.
4. Explore the charts and insights.
